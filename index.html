Nios2-llvm

LLVM based compiler for the software processor Nios 2 from Altera

Download this project as a .zip file Download this project as a tar.gz file

Welcome to GitHub Pages.

This project is intended to provide a full compiler tool-chain for the soft-core processor Nios II created by Altera

At the moment, it relies on Altera's assembler and linker.

Building the compiler

The compiler currently depends on Altera's assembler and linker.

Install gcc toolchain by Altera

The Nios II embedded design suite comes with gcc version 3.4.6
  1. Download the "Nios II Embedded Design Suite Legacy Tools" coming from Altera's website.
  2. Unpack it 
    
$ tar xvf 12.0_178_legacy_nios2_linux.tar.gz
  3. Install gcc toolchain 
    
$ sudo 12.0_178_legacy_nios2_linux/linux_installer/nios2eds/install
    By default, the package is installed in /opt/altera12.0
  4. Now, you need to set up binary paths 
    
$ export SOPC_KIT_NIOS2=/opt/altera12.0/nios2eds
$ export GNU_TOOLS_NIOS2=$SOPC_KIT_NIOS2/bin/nios2-gnutools
$ export PATH=$GNU_TOOLS_NIOS2/H-i686-pc-linux-gnu/bin:$PATH

Building LLVM compiler

The building process is quite simple. You will need to clone LLVM and its default c/c++ front-end clang:


$ git clone https://github.com/ivanllopard/nios2-llvm.git
$ cd nios-llvm/tools
$ git clone https://github.com/ivanllopard/nios2-clang.git clang

After that, create a build directory:


$ mkdir build

And compile/install 


$ cd build
$ ../nios2-llvm/configure --enable-targets=nios2 --target=nios2 --enable-optimized=yes
$ make && make install
